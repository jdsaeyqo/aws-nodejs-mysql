# 프로젝트 소개
- 제목 : 관심사 같은 친구 찾기
- 구분 : 개인 프로젝트
- 개발환경  :  
서버 - AWS EC2, NodeJS  
DB - AWS RDS, MySQL
- 사용 언어 - Kotlin
- 내용  
프로필 설정(닉네임, 나이, 직업, 관심사 3개)  
나와 관심사가 같은 사람들의 목록 표시  
서로 좋아요 누를 시 채팅 방 생성
